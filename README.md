# Ashwick Level 0 Character Sheet

A self-contained static character creation sheet for an Ashwick village-centered D&D campaign.

## Features

- Level 0 campaign rules and creation checklist
- Ability score controls with modifiers, HP, and point-buy estimate
- Ashwick occupation, village ties, future class path, and strange signs prompts
- Print, JSON export, clear, and local autosave controls

## Local Preview

Open `index.html` directly, or serve the folder:

```bash
python -m http.server 8787
```

Then visit `http://127.0.0.1:8787/`.

## Deploy

This is a static site. Vercel can deploy it from the repository root with no build command.
